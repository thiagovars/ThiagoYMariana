Bodas HTML Wedding Template

Documentation



"BODAS" is a Responsive HTML Wedding Template with clean, elegant and modern design. Great for wedding invitation or engagement. Coming with wedding features such as Couple Timeline, Countdown, Photo gallery, featured people, Gift Registry, RSVP and others. This template is responsive using Bootstrap Framework and optimized for mobile device.

Colors

Bodas Template comes with 5 altername color schemes: Pink (default), Green, Violet, Nude and Blue.

IMPORTANT:
When you change CSS color scheme it will not affect the COUNTDOWN colors, so you will need to modify it separately in js/theme-main.js file.

CSS color

To change theme, you need to use specified styles file instead of css/colors/pink.css.

RSVP Form

To edit RSVP Form php file, please open the file call send_form.php

  • You can change the email where the form data will be sent on line #3. $your_email="your_email@site.com";
  • You can change the ERROR ajax message after sent form on line #5. $result = '<img src="img/form-icon-error.png" style="width...
  • You can change the SUCCESS ajaxa message after sent form on line #24. $result = '<img src="img/form-icon-ok.png" style="width...

<?php
//Add here the email where you want to recieve data sent from the form.
$your_email="your_email@site.com";

$result = '<img src="img/form-icon-error.png" style="width: 90px;"/><br/><span class="title error">Sorry!</span><br/>Your data has not been sent. Please try again.<br /><strong>Error: #AJ002</strong><br /><br /><button class="btn btn-default BtnCloseResult" type="button">Close</button>';
if(!empty($_POST)){
    
    $events = '';
    if(isset($_POST['events']) && $_POST['events'] != ''){
        foreach($_POST['events'] as $value){
            $events .= ''.$value.', ';
        }
    }
    $name = $_POST['name'];
    $email = $_POST['email'];
    $message = $_POST['message'];
    
    $to      = $your_email;
    $subject = 'New RSVP Form: '.$name;
    $headers = 'From: '.$name.' <'.$email.'>' . "\r\n";
    $message = 'New RSVP data sent from your website:'."\r\n".'Name: '.$name."\r\n".'Email: '.$email."\r\n".'Will attend: '.$events."\r\n".'Message: '.$message;
    
    if(mail($to, $subject, $message, $headers)){
        $result = '<img src="img/form-icon-ok.png" style="width: 90px;"/><br /><span class="title success">Thanks!</span><br>Your info has been sent!<br /><br /><button class="btn btn-default BtnCloseResult" type="button">Close</button>';
    }
}

echo $result;
?>
